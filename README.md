## Key Features

* 2 types of User - Doctor & Patients
* Patients Features
  - Schedule Video calls with doctors 
  - Update and view Profile
  - Prescription Download
  - Health Articles
    
* Instructor Features
  - Add Details
  - Set Availability
  - Join video calls
  - Write Prescription for Patients after meet
